# Тестовое задание
```
Шаги по настройке проекта:
1. Выполнить команду: "npm i" (для установки зависимостей).
2. Для запуска проекта с тестом "Задание 1" требуется выполнить команду: "npm run test".
3. Для генерирования отчёта с тестом "Задание 1" требуется выполнить команду: "npm run reportUI".
4. Для запуска проекта с тестом "Задание 2" требуется выполнить команду: "npm run api".
5. Для генерирования отчёта с тестом "Задание 2" требуется выполнить команду: "npm run reportAPI".
```
```
Дополнительные файлы к заданиям находятся в папке "Report".
```
